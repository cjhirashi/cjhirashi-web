# Blog Components

Componentes específicos para el blog.

## Componentes a implementar:

- `post-card.tsx` - Card de preview de post
- `post-list.tsx` - Lista de posts con paginación
- `post-header.tsx` - Header de post individual
- `markdown-renderer.tsx` - Renderizador de markdown con syntax highlighting
- `table-of-contents.tsx` - Tabla de contenidos generada automáticamente
- `reading-progress.tsx` - Barra de progreso de lectura
- `share-buttons.tsx` - Botones para compartir en redes

Integra con CMS para renderizar contenido dinámico.
