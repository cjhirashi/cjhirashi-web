# UI Components

Componentes base reutilizables del design system.

## Componentes a implementar:

- `button.tsx` - Botones con variantes (primary, secondary, outline, ghost)
- `card.tsx` - Cards para contenido
- `input.tsx` - Inputs de formularios
- `textarea.tsx` - Textareas
- `select.tsx` - Selects
- `badge.tsx` - Badges/tags
- `alert.tsx` - Alertas (success, error, warning, info)
- `dialog.tsx` - Modales
- `dropdown.tsx` - Dropdowns
- `loading.tsx` - Loading spinners
- `skeleton.tsx` - Skeleton loaders

Todos los componentes siguen el design system definido en `/sys-docs/DESIGN-SYSTEM.md`
