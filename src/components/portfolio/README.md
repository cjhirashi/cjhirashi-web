# Portfolio Components

Componentes para mostrar proyectos del portfolio.

## Componentes a implementar:

- `project-card.tsx` - Card de proyecto con imagen, tech stack, descripción
- `project-grid.tsx` - Grid de proyectos con filtros
- `project-gallery.tsx` - Galería de imágenes de proyecto
- `tech-stack-badges.tsx` - Badges de tecnologías usadas
- `project-timeline.tsx` - Timeline de desarrollo del proyecto

Muestra proyectos profesionales con detalles técnicos.
