# Layout Components

Componentes de estructura y navegación.

## Componentes a implementar:

- `navbar.tsx` - Barra de navegación principal
- `footer.tsx` - Footer del sitio
- `sidebar.tsx` - Sidebar para admin dashboard
- `mobile-menu.tsx` - Menú móvil responsive
- `breadcrumbs.tsx` - Breadcrumbs para navegación

Layout adapta entre público (minimal, clean) y admin (sidebar, dashboard style).
